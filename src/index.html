Semantic UI is een HTML/CSS framework

Over Semantic UI

Semantic UI is een voorgemaakt CSS framework dat bedoelt is om het makkelijker te maken voor mensen om sites te maken.
Het is ontstaan toen men een nieuw framework wilde naast bootstrap. Een framework dat een iets groter leercurve had dan andere frameworks

Waarom hebben wij dit framework niet gekozen?

Semantic UI is een framework wat ontworpen is voor gevorderden of proffesionele front-end programmeurs. Op youtube en op google was weinig te vinden over Semantic UI. Waardoor wij niet in staat waren een mooie site te maken.
