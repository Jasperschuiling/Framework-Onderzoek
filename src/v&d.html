Voordelen en Nadelen

Op deze pagina beschrijven wij een aantal voordelen en nadelen van Semantic UI.

Voordelen

De voordelen van dit framework zijn:

Als je een developer bent die al veel kennis heeft van html / css is dit een zeer fijn framework om mee te werken. Je kan profesionele sites maken die erg mooi ogen.
Er zijn maarliefst 3000+ verschillende thema's. Natuurlijk zijn je websites responsive en werken op elk apparaat.

Nadelen

De nadelen van dit framework zijn.

Als je een beginner bent kan ik je dit framework afraden om te gebruiken. Niet alleen moet je meer kennis hebben van de programmeer talen, Ook is er weinig tot geen informatie te vinden op het internet. Volgens de site van Semantic UI zijn er op het ogenblik ongeveer 5000 commits met dit framework gedaan. Dat is niet erg veel en je zal waarschijnlijk vast lopen. Daarom raden wij aan om als eeste met bijvoorbeeld Bootstrap te gaan werken, Zodat je meer ervaring met deze frameworks krijgt.
